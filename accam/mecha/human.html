Human Body Structure

Cấu trúc cơ thể người
The human body is an amazing and complex structure composed of cells → tissues → organs → organ systems → the complete organism.

Cơ thể người là một cấu trúc kỳ diệu và phức tạp, được tạo thành từ tế bào → mô → cơ quan → hệ cơ quan → cơ thể hoàn chỉnh.

Main Parts of the Body

Head - Đầu
Head / Đầu
Contains: brain, eyes, ears, nose, mouth...
Neck - Cổ
Neck / Cổ
Connects head to torso, contains throat, trachea, esophagus...
Torso/Trunk - Thân
Torso / Trunk / Thân
Consists of: thorax (ngực) & abdomen (bụng)
Upper Limbs - Chi trên
Arms / Upper limbs / Tay - Chi trên
Shoulder → arm → elbow → forearm → wrist → hand
Lower Limbs - Chi dưới
Legs / Lower limbs / Chân - Chi dưới
Hip → thigh → knee → leg → ankle → foot

Major Organ Systems

Các hệ cơ quan chính trong cơ thể người (Major human organ systems)

  • Skeletal system — Hệ xương
  • Muscular system — Hệ cơ
  • Nervous system — Hệ thần kinh
  • Circulatory system — Hệ tuần hoàn
  • Respiratory system — Hệ hô hấp
  • Digestive system — Hệ tiêu hóa
  • Urinary system — Hệ bài tiết
  • Endocrine system — Hệ nội tiết
  • Reproductive system — Hệ sinh sản
  • Integumentary system — Hệ da (da, tóc, móng)
